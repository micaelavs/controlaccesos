    • INTRODUCCION

    La Dirección de Integración de Sistemas del Ministerio de Transporte de la Nación, ha diseñado un Sistema informático para el Acceso de Personas al Ministerio y sus distintas dependencias, permitiendo su implementación a otros organismos.

    ACCESO AL SISTEMA

    Para acceder al sistema de Presentismo, el usuario deberá ingresar a la siguiente dirección: intranet.transporte.gob.ar y despues ingresar al modulo Control de Accesos.

    Para ingresar al sistema, el usuario deberá usar su usuario y contraseña de dominio, una ves ingresado los datos podra navegar segun las credenciales que posea.

    ALTAS

    USUARIOS

    Este apartado permite dar de alta nuevos usuarios completando obligatoriamente los campos Nombre y Apellido y DNI; o dar de baja usuarios pre-existentes. Es condición excluyente para ser registrado como usuario, haber sido registrado previamente como empleado.

    Para ingresar a la pantalla de Registrar Usuario hacemos click en el boton Nuevo.
    Ingresaremos en la siguiente pantalla:

    Deberemos completar los campos Nombre de Usuario y Documento para poder asociar el usuario a los datos del empleado. Se podrá modificar los datos y el perfil de cada usuario, a fin de restringir o brindarle acceso a otras tareas.

    DEPENDENCIAS

    En esta seccion se podran visualizar el listado completo de las dependencias cargadas dentro del sistema.

    PERSONAS

    En esta seccion podremos visualizar el listado completo de las Personas dentro del sistema.

    Tambien agregar personas seleccionando el boton Agregar, deberemos completar los campos Documento, Nombre, Apellido y Genero.

    EMPLEADOS

    En esta seccion podremos visualizar el listado completo de los Empleados con sus correspondientes datos:

    • Documento
    • Nombre
    • Apellido
    • Ubicacion Principal
    • Ubicaciones Autorizadas
    • Dependencia Principal
    • Contrato
    • Enrolado

    Tambien contaremos con opciones de filtrado como Dependencias, Ubicaciones, Ubicacion Principal, Tipo de Contrato, Enrolado y un boton para poder visualizar el listado de empleados inactivos.

    Para dar de alta un Empleado deberemos seleccionar el boton Nuevo ingresaremos en una pantalla donde visualizaremos un formulario el cual deberemos completar ingresando todos los datos obligatorios como Documento, Nombre, Apellido, CUIT, Email, Genero, Dependencia Principal, Fecha Desde, Fecha Hasta, Cargo, Ubicaciones Autorizadas y Ubicacion Principal.

    Cada empleado contara en la pantalla principal con una columna llamada Acciones que contendra cuatro iconos Editar () Modificar Contrato () Borrar () y Horario ().
    Tambien contaremos con un exportador a Excel donde podremos visualizar todos los datos de los empleados. Tambien dispondremos de un boton para poder visualizar el listado de empleados Inactivos dentro del sistema y opciones de Filtrado.

    MODIFICAR EMPLEADO

    En este apartado podremos modificar los datos del empleado, la pantalla es igual a la que se nos presenta cuando damos de alta un nuevo empleado

    MODIFICAR CONTRATO

    En este apartado podremos modificar el contrato del empleado, ya sea finalizarlo o renovar el mismo

    BORRAR EMPLEADO

    Al selecciona este icono se podra dar de baja al empleado dentro del sistema, la condicion sera que no debera tener movimientos dentro del sistema para poder borrarlo/

    HORARIO

    En este apartado podremos visualizar el horario asignado del empleado. Tambien podremos asignarle una Plantilla Horaria o asignales los horarios manualmente en cada uno de los dias de la semana.

    UBICACIONES

    En este aparte podremos visualizar el listado de las Ubicaciones dentro del sistema en las cuales estaran asociados los diferentes empleados.

    Para agregar una nueva ubicacion deberemos seleccionar el boton Nuevo y completar los tres campos obligatorios Nombre, Calle y Numero.

    Tambien contaremos con dos iconos en la columna Acciones que son Editar () y Borrar () .
    En los cuales si seleccionamos el primero podremos editar los datos de la ubicacion y si selecciona el segundo procederemos a borrar la ubicacion, siempre y cuando la mismo no tenga empleados asociados.

    NOVEDADES

    En esta seccion podremos dar de alta las Novedades de los empleados, contaremos con opciones de filtrado y un listado completo con todas las novedades cargadas.
    Tambien contaremos con un exportador a Excel para tener la informacion mas detallada y podremos Editar () y Eliminar () las novedades.

    Para dar de alta una Novedad deberemos hacer click en el boton Nuevo, deberemos ingresar el documento del empleado, seleccionar el tipo de novedad que se le asignara y la fecha desde/fecha hasta.
    Tambien contaremos con un listado en la parte inferior de la pantalla con todas las novedades que fueron ingresadas para el empleado y un exportador a Excel de las novedades propias del empleado. En esta misma pantalla se podran editar las novedades de ser necesario.
    Para finalizar el proceso seleccionamos Guardar

    HORARIOS

    En esta seccion podremos registrar una banda horaria, tambien podremos Editar (), Eliminar () y realizar una Carga Masiva () para la misma.

    Para poder registrar una banda horaria deberemos hacer click en el boton Nuevo y completar los datos Nombre, Rango de dias y Rango de Horario.
    Para finalizar hacemos click en Guardar

    CARGA MASIVA

    Dentro de este apartado podremos realizar una carga masiva de una planilla horaria para los agentes de una dependencia especifica, para asi de esta forma no tener que realizarlo de una forma individual.

    Podremos Guardar o Guardar y Continuar.

    REGISTRO

    CARGA OFFLINE PARA EMPLEADOS

    En este formulario podremos realizar una carga manual para los empleados completando los campos obligatorios como Ubicacion, Fecha, Documento, Nombre y Apellido del Empleado, Horario de Ingreso y Horario de Egreso. Tambien se podran cargar observaciones.

    REPORTES

    EMPLEADOS

    En este apartado se podrá obtener la información que se requiera respecto a los empleados, Contaremos con un filtro desplegable en donde podremos filtrar por Ubicacion, Dependencia, Fecha Desde, Fecha Hasta y Otros Criterios, para realizar la busqueda debemos hacer click en el boton Filtrar.
    En el listado cada tipo de Ingreso y Egreso estaran diferenciados por:

    • Registro de Comision Horaria
    • Registro desde puesto CIET
    • Registro Biometrico
    • Registro Biometrico MECON
    • Registro Manual
    • Sin Registro
    PLANILLA RELOJ

    En esta seccion podremos Descargar la planilla que contiene los horarios de los empleados, deberemos ingresar la Dependencia y la Fecha y seleccionar el boton Planilla Otras Modalidades o Planilla Ley Marco.

    REPORTE DESVIOS DE HORARIOS

    En esta seccion podremos generar el informe de desvios de horarios, deberemos completar un formulario ingresando la Dependencia, Tipo de Contrato, Fecha Desde y Fecha Hasta. Tambien podremos realizar un informe de un empleado especifico que necesitemos ingresando el Documento.
    Para generar el informe seleciconamos el boton Generar Informe.

    CALENDARIO PLANILLA

    En esta seccion podremos consultar la Planilla Calendario mensual, en la misma podremos verificar los datos de Presentes, Ausentes y Novedades de los empleados.
    Para realizar la consulta deberemos ingresar la Dependencia, el Contrato los cuales podremos seleccion individualmente o todos los contratos haciendo click en la opcion Seleccionar Todos y la Fecha la cual sera solo el Mes y Año. Una vez ingresados los datos seleccionaremos el boton Consultar para cargar el Calendario.

    AUSENTISMO ONLINE

    Dentro de este apartado podremods visualizar en tiempo real la cantidad de ausentes de cada uno de las dependecias de los tres tipos de contratacion Ley marco, Planta Permanente, Asistencia Tecnica y poder verificar el Total General.

    CONFIGURACION INFORME MENSUAL

    En este apartado podremos generar un informe mensual de manera manual de la dependencia que seleccionemos y de los datos de los tipos de Contratos que contenga la misma.

    Para dar de alta una Configuracion de Informe Mensual deberemos seleecionar el boton Nuevo y completar los campos obligatorios del formulario en pantalla.

    Tambien contaremos con opciones en la columna Acciones que son Editar () , Borrar () y Envio Mensual () , este ultimo permitira enviar un mail al empleado en el cual se encuentre configurado el Informe Mensual con todos los datos de dicho informe.

    ADMINISTRACION

    SOLICITUDES

    En este apartado podremos visualizar todas las Solicitudes que se realizen dentro del sistema y administrar las mismas.

    MIS CONSULTAS

    MIS HORARIOS

    En este apartado se podra visualizar el horario del usuario logueado.

    CERRAR SESIÓN

    Para salir del sistema el usuario debe CERRAR SESIÓN en el botón que se encuentra en la esquina superior derecha.
